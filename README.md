Rock Paper Scissors Lizard Spock — Ultimate Edition

📖 About the Game
This is not your regular Rock Paper Scissors! This Ultimate Edition adds Lizard and Spock to the mix (made famous by The Big Bang Theory 🖖), giving you 10 unique rules and a much more exciting game.
You play against an AI opponent that actually learns from your moves — the harder the difficulty, the smarter it gets!

✨ Features
```text
🖖 5 choices — Rock, Paper, Scissors, Lizard, Spock
🤖 3 AI difficulty levels — Easy, Medium, Hard
📊 Live score tracker — Wins, Losses, Ties, Win Rate
💾 Leaderboard — saves your all-time scores
🎨 Dark neon UI — built with Streamlit + custom CSS
⚡ Instant play — no login, no download needed
 ```

# Clone the repo
git clone https://github.com/stackpilot24/Rock-Paper-Scissors.git
cd rock-paper-scissors

# Terminal version (no install needed)
python game.py


# Web UI
```text
pip install streamlit
streamlit run streamlit_app.py
```
